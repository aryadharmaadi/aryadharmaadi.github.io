Short description about Fuzzing:

Fuzzing is a dynamic software testing technique running automatically for finding software vulnerabilities. We call it dynamic because, instead of just scanning the source code, we must run the software under test (SUT) to watch the software behaviour. The main idea of the fuzzing is producing tons of test cases, injecting them into the SUT, and observing the behaviour, which is done automatically.
